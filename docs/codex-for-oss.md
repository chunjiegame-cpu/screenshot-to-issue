## Why This Repository Qualifies

Screenshot to Issue is an early-stage maintainer tool for improving visual bug triage. It turns screenshots into structured GitHub issue drafts with metadata, reproduction prompts, expected behavior, actual behavior, and a checklist, reducing follow-up work for UI maintainers.

## API Credit Usage Plan

Use API credits to prototype Codex-assisted issue triage from generated drafts, suggest labels and reproduction steps, summarize screenshot bug clusters, and draft maintainer responses that ask for only the missing context.
